# Org Theory Syllabus Repository

A community-sourced collection of organizational theory syllabi from education scholars across universities.

## About

This repository collects syllabi from organizational theory courses taught in education programs and related departments. It is intended as a resource for instructors designing or revising their own courses.

**Live site:** [View the repository](https://YOUR-USERNAME.github.io/org-theory-syllabus-repo/)

## What's Included

The repository currently contains syllabi from courses at institutions including:

- UC Berkeley
- Northwestern University
- University of Cincinnati
- University of Maryland
- University of Pittsburgh
- University of Utah
- University of Virginia
- Vanderbilt University
- Washington State University

Courses span undergraduate through doctoral levels, covering topics like organizational theory, organizational behavior/change, leadership/management, policy implementation, and sociology.

## How to Use

Visit the website to browse, search, and filter courses by focus area, modality, student level, or institution. Each course entry links to the contributed syllabus document.

If you use a syllabus from this collection, please credit the contributor as requested.

## Contributing

Want to add your syllabus? [Submit through our Google Form](https://forms.gle/DfdkPQqzZzwfKuQR7).

## Setup

This is a static site designed for GitHub Pages. To deploy:

1. Create a new GitHub repository
2. Push this folder's contents to the `main` branch
3. Go to **Settings > Pages** and set the source to `main` branch, root (`/`)
4. Your site will be live at `https://YOUR-USERNAME.github.io/REPO-NAME/`

## Data

Course data is stored in `data/syllabi.json`. To add or update entries, edit that file and commit.
